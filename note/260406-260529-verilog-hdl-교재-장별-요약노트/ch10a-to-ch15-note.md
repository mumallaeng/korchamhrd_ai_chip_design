# PART 02-03 메모

PART 02와 PART 03은 실제 회로 모델링과 FPGA 구현으로 이어지는 구간이다.

현재 강의 범위와 가장 직접적으로 맞닿는 부분:

- CH10 조합회로 모델링
- CH11 순차회로 모델링
- CH12 실습장비
- CH13 Vivado 사용법
- CH14 조합회로 구현 실습 일부
- CH15 순차회로 구현 실습 일부

지금 학습 전략상 우선순위는 다음과 같다.

1. CH10, CH11을 강의 노트와 구현 예제로 고정
2. CH12, CH13을 실습 흐름과 XDC/툴 사용으로 고정
3. CH14, CH15를 작은 보드 과제로 재구성

읽을 때는 `문법 -> 회로 -> 시뮬레이션 -> 구현 -> 보드 동작`으로 연결되는지를 계속 확인한다.
