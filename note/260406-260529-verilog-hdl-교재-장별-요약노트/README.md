## 목적

이 폴더는 `Verilog HDL 설계 - Vivado와 FPGA를 이용한 설계 실습`의 장별 내용을, 교재 원문 없이도 다시 학습할 수 있도록 재구성한 강의 노트 모음이다.

기준은 다음과 같다.

- 교재 장 제목과 공개적으로 확인 가능한 목차
- 실제 수업 메모와 복습 노트
- 공개 참고 자료
- 현재 수업에서 사용하는 `Basys3 + Vivado` 실습 문맥

## 작성 원칙

- 책 문장을 옮기지 않는다.
- 개념 설명 뒤에 `문법만 먼저 보기` 또는 `형식만 먼저 보기`를 둔다.
- 그 다음에 전체 예제를 둔다.
- 예제 뒤에는 반드시 해석과 실전 포인트를 붙인다.
- 교재 내용과 현재 수업 연결점을 따로 적는다.
- 교재 기반 노트는 가능한 한 공식 목차 계층을 보존한다.
- 책 전체 인덱스에서는 `# PART`, `## CHAPTER` 구조를 유지한다.
- 개별 chapter note에서는 `# CHAPTER 제목`, `## 공식 절 제목`, `### 확장 설명` 순서를 우선한다.
